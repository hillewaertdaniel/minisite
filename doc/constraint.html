Mini-Site

Contraintes techniques

  • Le site sera fonctionnel sur smartphones/tablettes/bureau
  • A vous de prévoir les points de rupture
  • Vous ne pouvez pas utiliser de frameworks existants style boostrap, fundation, 960, ...A vous de créer vos positonnements
  • Le site sera Mobile First
  • L'unité de mesure utilisée des polices sera le rem
  • Utilisez le box-sizing
  • Utilisez la technique de positionnement FLEXBOX pour la structure générale. Vous pouvez bien sûr utiliser toutes les autres techniques de positonnement pour les positonnements internes
  • La taille max en vue bureau sera de 1160px
  • La vue mobile est imposée pour la page d'acceuil, pour les suivantes à vous de proposer des interfaces fonctionnnelles
  • A la racine de votre espace minimir, créer une page d'index reprenant un lien vers votre projet.
  • Le dossier reprenant votre projet sera nommé minisite
